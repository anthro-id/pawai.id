# PAWAI 2026: Gods Paradise
This is an official GitHub repository of a PAWAI (pawai.id) website, maintained by Anthro Indonesia developer/s.

## Techs
We are using [Next.js (Page Router)](https://nextjs.org) (v16.x) with [Node.js](https://nodejs.org) (v24.x) engine, hosted in [Vercel](https://vercel.com).
